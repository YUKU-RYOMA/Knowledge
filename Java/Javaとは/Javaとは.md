
### オブジェクト指向言語

**オブジェクト指向**とは、アプリケーションの中で扱う対象を「モノ」に見立て、その組み合わせによって、機能を作り上げていく手法。

### JavaScriptとは異なる言語

名前が似ているせいで混同されがちだが、全くの別物。
Javaは、Java専用の仮想マシン上で動作するが、JavaScriptは、主にブラウザ上で利用する簡単なスクリプト言語。


### プラットフォームに依存しない

Javaのプログラムは、**Java仮想マシン**で動作する**中間コード**に変換され、動作する。
プラットフォーム固有のネイティブコードを生成しないため、Javaで開発されたアプリケーションが特定のプラットフォームに依存することがない。

プラットフォームに対応した、Javaの仮想マシンさえあれば、Javaで書かれたプログラムはどこでも動作する。
これを**Write Once, Run Anywhere**という。


### ガベージコレクタがメモリを自動で管理

Javaでは、**ガベージコレクタ**と呼ばれる仕組みが用意されていて、使われなくなったメモリを自動的に解放してくれる。
メモリの解放を忘れてメモリリークを起こさせることが原則として発生しない。
